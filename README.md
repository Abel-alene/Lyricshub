# 🎵 Lyrics Hub Bot

A simple Telegram bot that fetches song lyrics instantly.

Created by **Abel Alene** - [@xoiam](https://t.me/xoiam)

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

---

## 📦 Quick Setup

### 1. Clone the repository
```bash
git clone https://github.com/Abel-alene/Lyricshub
cd lyricshub
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run setup

```bash
python setup.py
```

4. Start the bot

```bash
python run.py
```

---

🔧 What You Need

· Python 3.8+ installed
· Bot Token from @BotFather on Telegram

---

📱 How It Works

1. Send any song name to the bot
2. Choose from matching results
3. Get instant lyrics with album art

---

🔧 Commands

Command What It Does
/start Welcome message
/help Help guide
song name Search for lyrics
